💡 Inline Editing Instructions

Hoe inline editing werken:

  • 🖱️ Klik op een veld (naam, telefoon, email, afdeling) om inline te bewerken
  • ✏️ Blauw edit knop opent volledige edit form inline
  • ✅ Groen vinkje = opslaan
  • ❌ Rood kruisje = annuleren
  • 🎯 Geen popups - alles gebeurt inline volgens Antwerp stijl
