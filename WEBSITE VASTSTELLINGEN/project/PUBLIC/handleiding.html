README.md - Instructions for setup and deployment # Antwerpen Vaststellingen Beheer Systeem Dit systeem bestaat uit een vaststellingsformulier en een beheerdersinterface voor het bijhouden van vaststellingen van nutswerken in Antwerpen. ## Bestanden - `vaststelling_formulier.html` - Het hoofdformulier voor het registreren van nieuwe vaststellingen - `index.html` - De beheersinterface om de keuzemogelijkheden te beheren - `data.json` - Het gegevensbestand dat alle configuratie opslaat - `server.js` - De Node.js server die alles aan elkaar koppelt ## Installatie 1. Installeer Node.js als je dat nog niet hebt (https://nodejs.org/) 2. Clone of download dit project naar je computer 3. Open een terminal in de projectmap 4. Installeer de nodige dependencies: ``` npm install express multer body-parser ``` 5. Maak een `public` map en plaats de volgende bestanden daar: - vaststelling_formulier.html - index.html - data.json 6. Start de server: ``` node server.js ``` 7. Open je browser en ga naar http://localhost:3000/ ## Gebruik ### Vaststellingsformulier - Het formulier is toegankelijk via http://localhost:3000/ - Alle keuzemogelijkheden worden automatisch geladen uit data.json - Ingevulde formulieren worden opgeslagen in data/vaststellingen.json - Geüploade foto's worden opgeslagen in de map uploads ## Beheerinterface - De beheerinterface is toegankelijk via http://localhost:3000/admin - Hier kun je de volgende gegevens beheren: - Vaststellers (medewerkers die vaststellingen uitvoeren) - Postcodes - Types nutsmaatschappijen - Nutsmaatschappijen - Alle wijzigingen worden automatisch opgeslagen in data.json ## Mapstructuur ``` project/ ├── public/ │ ├── vaststelling_formulier.html │ ├── index.html │ └── data.json ├── data/ │ └── vaststellingen.json (automatisch aangemaakt) ├── uploads/ (automatisch aangemaakt) ├── server.js └── package.json ``` ## Technische Details Het systeem is opgebouwd met: - HTML/CSS/JavaScript voor de frontend - Node.js/Express voor de backend - Gegevensopslag in JSON formaat - Pico CSS voor styling Het is een eenvoudig maar effectief systeem voor het bijhouden van vaststellingen zonder complexe database-installatie.
