📋 Component Beschrijving

OpbraakEnMateriaal.tsx is een React + TypeScript component gebouwd volgens de Antwerpse digitale huisstijl (@a-ui/react).

Features:

  • ✅ 2 Required checkboxes: "Werd natuursteen ingeslepen?" en "Werd asfalt ingeslepen?"
  • ✅ 1 Required select: "Werden materialen veilig gestockeerd?" met OK/NOK opties
  • ✅ Volledige toegankelijkheid (ARIA attributes, semantic HTML)
  • ✅ TypeScript types en client-side validatie
  • ✅ Inline error handling na blur events
  • ✅ A-UI styling met Antwerpen brand kleuren (#E10019)
  • ✅ Responsive design voor mobile en desktop
  • ✅ Fieldset/legend structuur voor form grouping

Gebruik:

import OpbraakEnMateriaal from './OpbraakEnMateriaal'

function App() {
  return (
    <div>
      <OpbraakEnMateriaal />
    </div>
  )
}
