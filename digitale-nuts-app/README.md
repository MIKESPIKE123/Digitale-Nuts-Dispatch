# Digitale Nuts App

## Versiebeheer
- Appversie: `v1.4`
- Bron van waarheid: `src/version.ts`
- Laatste update document: 8 februari 2026

## Overzicht
Digitale terreinapp voor nutsinspecties (Stad Antwerpen), met focus op:
- verplichte NOK-verantwoordelijke flow
- GPS + adresautomatisatie met fallback
- foto-evidence per element
- export naar PDF/CSV
- configureerbare inspecteurs, postcodes en werkpartijen

## Snelstart
```bash
npm install
npm start
```

Standaard start op `http://localhost:3001`.
